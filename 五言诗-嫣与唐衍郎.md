# 杯盘狼藉
## 乙巳年六月

...

欢歌不曾休，乘风过柴门。
柴门无所有，那曾闻仙音。
可怜机杼女，勤织王孙裳。

日月随流水，霜星更飞度。
学书深宫里，薰黛正及笄。
礼官报天子，吉日值庚寅。
天子准礼已，抚须犹慨叹。
适彼吉隆日，公主起华妆。
左右为调粉，对镜点朱丹。
青笔画蛾眉，鲜脂拭花黄。
相与束罗裙，金簪步摇长。
遂取云凤冠，华容正相当。
妆成时辰到，拥戴出闺房。
典堂三百步，步步绝雁郎。

...

典官宣唱已，上殿龙武卫。
昭昭鱼鳞铠，泛泛雁翎刀。
半跪合抱拳，从此随夕朝。
中有唐衍郎，寒门旧候子。
武高智双全，选作百夫长。

...

中门已失火，神器恐不测。
唤来亲侍卫，护与突重围。
按剑出宫门，相扶上马背。
自知九死易，抛主潜更难。
三五但相对，默默无有言。
抽刀擎斧锏，策马向西门。
官军业已溃，义兵不可当。
杀声动云天，刀光映萧墙。
华灯见斧钺，红颜哭号长。
朱门并漆柱，一炬付焚土。
公卿与王孙，弃尸中京路。
乱军乱人马，冷箭冷刀光。
兵戈相交鸣，人喊马嘶驰。
向来单八骑，披坚各执锐。
及至出西门，匹马只二人。
公主掩面泣，唐郎但无言。

杀声犹未远，追兵或即到。
卸甲弃于道，解挂留道边。
步摇掷下地，华襦委与田。
独藏黄金钗，会时自了结。
