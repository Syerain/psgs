# 《典堂三百步》 - 乙巳年六月 

...

欢歌不曾休，乘风过柴门。

柴门无所有，那曾闻仙音。

可怜机杼女，勤织王孙裳。

...

日月随流水，霜星更飞度。

学书深宫里，薰黛正及笄。

礼官报天子，吉日值庚寅。

天子准礼已，抚须犹慨叹。

适彼吉隆日，公主起华妆。

左右为调粉，对镜点朱丹。

青笔画蛾眉，鲜脂拭花黄。

相与束罗裙，金簪步摇长。

遂取云凤冠，华容正相当。

妆成时辰到，拥戴出闺房。

典堂三百步，步步绝雁郎。

...

典官宣唱已，上殿龙武卫。

昭昭鱼鳞铠，泛泛雁翎刀。

半跪合抱拳，从此随夕朝。

中有唐衍郎，寒门旧候子。

武高智双全，选作百夫长。

...

荧惑守心宿，流彗生干戈。

...

中门已失火，神器恐不测。

咸集亲侍卫，护与突重围。

按剑出宫门，相扶上马背。

自知九死易，抛主潜更难。

三五但相对，默默无有言。

抽刀擎斧锏，策马向西门。

官兵倾颓已，大盗不可当。

杀声动云天，刀光照萧墙。

华灯见斧钺，红颜泣涕长。

朱门并漆柱，一炬付焚土。

公卿与王孙，弃尸中京路。

乱军乱人马，冷箭冷刀光。

兵戈相交鸣，人喊马嘶驰。

向来单八骑，披坚各执锐。

及至出西门，匹马只二人。

公主掩面泣，唐郎但无言。

杀声犹未远，追兵或到即。

卸甲弃于道，解挂留路边。

步摇掷下地，华襦委与田。

独藏黄金钗，会时自了结。

中野单骑薄，策马去城远。

城远得山庙，且避夜连绵。

芙蓉凭案枯，雁刀倚壁冷。

海棠枕袖已，须眉犹守夜。

中夜诚良久，东方乃平旦。

晨起迎新曙，疏木荒红苕。

...

# 白话译文 (未更新，大幅度遗漏)

欢快的歌声不曾停歇，乘着风飘过简陋的柴门。
柴门里一无所有，哪曾听过仙乐般的曲调？
可怜的织布女子，辛勤纺织，只为王孙贵族的衣裳。

日月如流水般消逝，星辰飞转，光阴匆匆。
公主在深宫里读书习字，熏香染黛，正值及笄之年。
礼官上报天子，吉日选定在庚寅这天。
天子准了礼仪，抚须感叹，似有深意。

到了那吉日良辰，公主盛装打扮。
侍女们为她敷粉描眉，对镜点染朱唇。
青黛勾画蛾眉，胭脂涂抹额黄。
众人帮她系好罗裙，金簪步摇垂落，摇曳生姿。
最后戴上云凤冠，华美的妆容正相配。
妆成之时已到，众人簇拥她走出闺房。
通往典礼殿堂的台阶有三百步，每走一步，都让仰慕她的男子黯然失色。

……

典礼官宣读完毕，殿上站满龙武卫。
他们铠甲如鱼鳞闪烁，腰佩雁翎刀寒光凛凛。
他们半跪抱拳行礼，宣誓从此追随公主左右。
其中有个叫唐衍的郎将，出身寒门，曾是没落侯爵之子。
他武艺高强，智勇双全，被选为百夫长。

……

宫门突然燃起大火，皇权危在旦夕。
天子急召亲卫，命令他们保护公主突围。
唐衍按剑护驾，扶公主上马。
他心知自己战死容易，但若弃主潜逃，比死更难。
仅剩的三五亲卫相对无言，沉默肃杀。
他们抽出刀斧长锏，策马冲向西门突围。

官军已然溃败，起义军势不可挡。
喊杀声震天，刀光映照宫墙。
华丽的宫灯下，刀斧寒光刺眼，红颜女子哭嚎逃命。
朱门漆柱，尽数被大火吞噬。
王公贵族横尸京中大道。

乱军冲杀，人马混乱，冷箭横飞，刀光无情。
兵器碰撞，人喊马嘶。
原本八名精锐护卫拼死突围，可到了西门，只剩公主和唐衍两人。
公主掩面哭泣，唐衍沉默无言。

身后的杀声仍未远去，追兵随时可能赶上。
唐衍脱下铠甲丢弃路边，解下华贵外袍扔在田间。
公主摘下步摇掷于地上，脱下华服任其委弃尘土。
唯独藏起一支黄金发钗，倘若被俘，便用它自我了断。
